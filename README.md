## Usage

```sh
yarn add eslint @muhmdraouf/eslint-config-react-native --dev
```

In `.eslintrc`:

```json
{ 
  "extends": "@muhmdraouf/eslint-config-react-native"
} 
```

In `tsconfig.json`:

```json
{
  "extends": "@muhmdraouf/eslint-config-react-native/tsconfig"
}
```
